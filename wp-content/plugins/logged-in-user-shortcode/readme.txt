== Description ==
Allows you to easily display different content to users who are logged in and those who are logged out of your site.

Example Usage:
[loggedin]content[/loggedin] returns content only to logged in users
[loggedout]content[/loggedout] returns content only to logged out users

== Installation ==

1. Just use the "Add New" button in Plugin section of your WordPress blog's Control panel. To find the plugin there, search for 'Logged in User Shortcode'
1. Activate the plugin 
1. Place the shortcode as needed in your posts or pages

== Frequently Asked Questions ==

= Why is my formatting messed up within the shortcode? =

You will need to place the short code on separate lines to ensure your content is wrapped in paragraph tags:
[loggedin]
content
[/loggedin]
